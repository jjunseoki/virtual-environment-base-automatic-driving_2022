# FOSCAR_VEAC_2022

## 대회 내용
- 자율주행 프로그래밍 교육 제공(ROS, 시뮬레이션 활용법, GPS 센서, 경로추종 등)
- 가상환경에서 도로주행, 장애물 및 신호등 인식 자율주행 미션 수행
- 3차원 가상 주행로에서 트랙 기반 자율주행 경진대회
- ROS 환경에서 자율주행 알고리즘 프로그래밍
